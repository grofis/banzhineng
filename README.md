### 网站成立的第4️⃣天，离网站注销∞天。 

今天有好多收获，没忍住，有注册了一个新域名。

听了好几期播客，有几期知识密度挺高，让我重新把注意力放到AI出海和人群画像上来。

在开发上也取得了一定的进展，这还得从昨天想到的React和Vue的区别谈起。

尤雨溪不容易，从0到1开发了个那么好用，在前端领域那么能打的框架！形成了和React、Angular三足鼎立之势。

<img src="https://banzhineng.com/img/react.png" width="750" alt="React框架" style="vertical-align:middle; margin-right:10px;"> 

但是三个框架又各有侧重，比如现在最厉害的还是React，而且国内的大厂也倾向于使用React的趋势。

Vue的优势就是轻量化，从数据到见到产品时效性非常高，熟手一天就能做成几十个页面和组件。

然并卵，就像国内各种社交平台上充斥的电子垃圾一样，他们除了耗费了一点作者的精力之外，其实并没有赚到钱。

但这也暗合了亚洲人急躁，内卷，爱折腾的特质。不过能有缓解焦虑的作用，也不错了。

Angular就不太熟悉，我理解的是偏向于实现Android平台手机端的框架，说到Google和Android，

又不得不提一嘴Java付费，Android开发的官方语言从Java转变成了Kotlin，实属无奈之举。

而甲骨文为什么要对Java下手呢？本质上还不是赚不到钱了呗，这和世道变难了，又走到一块儿去了。
