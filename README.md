### 网站成立的第5️⃣天，离网站注销∞天 

今天了解了不少SEO的想法，对SEO的认识也更成熟了一点。

简单地说SEO就是怎样让你潜在的用户快速地找到你，你有那么多竞争对手，凭什么是你而不是你的对手呢。

这就是SEO做得好了！SEO有点类似于投机取巧，但又不只是投机取巧。

很多人认为AI都能自动生成内容了，SEO肯定不难而且完全可以自动化。但SEO难的可能不是生成内容，

难的是你生成的内容有很多人看，很多人愿意为你提供的服务买单。说白了，SEO就是辅助你生产内容，服务你的客户，

吸引更多潜在用户来你这里消费的工具。所以用机器优化SEO的这个命题本质上并不存在，始终都是需要人参与的。

### 网站成立的第8️⃣天，离网站注销∞天

今天来谈谈如何实现一个好产品？我认为想要打造一个产品，有以下五个步骤。

1、寻找，找到市面上优秀的，符合你品位的产品。值得注意的是符合你品位的产品，未必优秀。

2、获得，通过购买或者爬虫之类的黑科技获得优秀产品的服务。

3、享受，一个产品优不优秀其实和你能不能享受这个产品没有太多的关系。一个很烂的产品你同样也可以享受它，比如洋芋。

它既没有什么新鲜，刺激的口感，甚至吃起来平淡无味，但这并不妨碍很多人把它当做美味进行享用，比如添加了各种香味的薯片。

4、达到，就是当你爱一个产品爱到如痴如醉，如癫如狂地时候，你就达到了一种忘我的状态。

在这种状态下，你原来是什么，你将来会是什么都已经无关紧要了，那种状态下最重要的是你想什么，做什么，以及你的感觉是什么。

5、实现，其实太过于沉浸于某种状态，你也会觉得有点腻，比如那些整天冥想和部分时间冥想，部分时间行动，以及想冥想就冥想，想行动就行动的人。

有什么区别？ 其实就是个自由支配自己的身体，节奏，自由掌控身体的状态的一种区别。

整天冥想的人未必能取得最佳状态，因为你啥时也不干，大脑和身体都处于一种放空的状态，能取得啥效果，能取得啥进展呢？

而且身体只处于一种状态之下，是违反客观规律，生理规律的，比如整天大太阳，是不是要晒死了？整天整天下雨，云南就要出蘑菇🍄，人也要发霉了。

部分冥想，部分行动的人会好一些，因为他们既有思考，又有行动，还有反馈，如果再添加进一点复盘和优化，那就更好了。

想冥想就冥想，想行动就行动，那是实现了财富自由的人干的，一般人做不到。
